# Bank-Financial-Analysis-Excel-Model
Excel-based banking analysis using financial statements, profitability metrics, margins, yields, and return on equity calculations.

## Project Overview
This project analyzes the financial performance of a commercial bank using Excel. The model uses balance sheet and income statement data to calculate bank revenues, earnings before taxes, net income, margins, yields, and return on equity.

The purpose of this project is to demonstrate banking financial statement analysis, profitability analysis, and Excel-based financial modeling skills.

## Tools Used
- Microsoft Excel
- Financial Statement Analysis
- Banking Analysis
- Profitability Analysis
- Margin and Yield Calculations
- Return on Equity Analysis

## File Included
- `bank_financial_analysis_model.xlsx`: Excel model containing the bank financial statement analysis and calculated outputs.

## Analysis Summary
The model evaluates how the bank generates revenue from different business areas, including securities, enterprise loans, consumer loans, and equity participations. It also breaks down funding costs from bonds issued, syndicated loans, and deposits.

The analysis connects balance sheet items with income statement performance to understand how assets, liabilities, revenues, expenses, and risk costs affect the bank’s profitability.
